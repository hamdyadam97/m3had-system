# نظام إدارة المعاهد

نظام متكامل لإدارة الحركة اليومية لفروع المعاهد التعليمية، يتيح متابعة الإيرادات والمصروفات والطلاب والدورات بشكل فعال.

## المميزات الرئيسية

### 📊 إدارة الفروع
- إدارة 16 فرع بشكل منفصل
- كل فرع له بياناته وصلاحياته الخاصة
- تحديد أهداف شهرية لكل فرع (افتراضي: 700,000 ج.م)

### 👥 إدارة المستخدمين
- **مدير النظام**: صلاحيات كاملة على جميع الفروع
- **مدير الفرع**: صلاحيات على فرعه فقط
- **الموظف**: إدخال البيانات والتحصيلات
- **المحاسب**: متابعة التقارير المالية

### 📚 إدارة الدورات والدبلومات
- إضافة دورات ودبلومات بأسعار مختلفة
- ربط الدورات بالفروع
- متابعة عدد المسجلين والإيرادات لكل دورة

### 🎓 إدارة الطلاب
- تسجيل طلاب جدد
- دفع كاش أو بالتقسيط
- متابعة الأقساط والمبالغ المتبقية
- طرق دفع متعددة (كاش، فيزا، تحويل بنكي)

### 💰 المعاملات المالية

#### المدخلات (الإيرادات)
- **تسجيل جديد**: دفعات الطلاب الجدد
- **تحصيل**: أقساط الطلاب المسجلين سابقاً
- تسجيل طريقة الدفع ومكان الدفع
- معرفة الموظف الذي حصل المبلغ

#### المخرجات (المصروفات)
- رواتب
- إيجار
- مرافق (كهرباء، ماء، إنترنت)
- مستلزمات
- تسويق
- صيانة

### 📈 التقارير

#### التقرير اليومي
- إجمالي الإيرادات والمصروفات
- تفاصيل التسجيلات والتحصيلات
- حسب طريقة الدفع (كاش، فيزا، تحويل بنكي)
- حسب مكان الدفع (حضوري، عن بعد)

#### التقرير الشهري
- مقارنة التحصيل الفعلي بالهدف
- نسبة الإنجاز
- الربح الصافي
- إحصائيات الطلاب

#### تقرير الدورات
- إحصائيات كل دورة/دبلومة
- عدد المسجلين والإيرادات
- يومياً وشهرياً

#### تقرير الموظفين
- أداء كل موظف
- إجمالي المحصل
- حسب طريقة الدفع

## التثبيت والتشغيل

### المتطلبات
- Python 3.10+
- Django 5.0+

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
cd institute_management
```

2. **تثبيت المتطلبات**
```bash
pip install django
```

3. **إنشاء قاعدة البيانات**
```bash
python manage.py makemigrations
python manage.py migrate
```

4. **إنشاء البيانات التجريبية (اختياري)**
```bash
python create_demo_data.py
```

5. **تشغيل الخادم**
```bash
python manage.py runserver
```

6. **فتح المتصفح**
```
http://127.0.0.1:8000/
```
